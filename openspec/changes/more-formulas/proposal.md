# Proposal: More Formula Functions

## Summary
Add more pixel formula functions to PixelFormulaEngine for richer visual output.

## Current Formulas
- `BAR` — Progress bar
- `TEXT` — Text label
- `STATUS` — Status indicator
- `BOX` — Box outline
- `SPARKLINE` — Mini chart

## New Formulas
| Formula | Purpose |
|---------|---------|
| `RECT` | Filled rectangle |
| `LINE` | Horizontal/vertical line |
| `CIRCLE` | Circle (filled or outline) |
| `GAUGE` | Circular gauge/arc |
| `NUMBER` | Formatted number display |
| `TIME` | Current time display |

## Impact
- Richer visual dashboards
- More flexibility for AI agents
- Better data visualization
